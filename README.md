# freza-plus-frontend

### Фронт сайта компании FrezaPlus.

Проект информационного сайта, на котором представлены услуги и цены компании. 
Приложение сайта получает информаци с API сервера:
* GET /routes - сореджимое основных страниц
* GET /adress - адреса компании и контакты
* GET /services - содержимое страниц услуг компании

Проект не является законченным, и находится в процессе разработки.

## Запуск проекта

1. Клонируем репозиторий
  ```
  git clone git@github.com:Buzhak/freza-plus-frontend.git
  ```
2. Устанавливаем зависимости.
  ```
  npm install
  ```
3. Запускаем проект.
  ```
  npm start
  ```
4. Зпуск json-server, на 5000-м порте
   ```
   json-server ./src/db.json --port 5000
   ```
